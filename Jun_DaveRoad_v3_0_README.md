```

manifest.json           ← PWA設定

```

**全4ファイルを同じフォルダに配置する必要あり**。

---

## 🚀 インストール手順

### Android（Chrome）
1. 4ファイルを端末の同じフォルダに保存
2. ローカルHTTPサーバ経由でアクセス（127.0.0.1:11219 等）
3. Chrome メニュー → 「ホーム画面に追加」
4. アイコンタップでアプリ起動

### iOS（Safari）
1. 同上、4ファイルを配置
2. Safari でアクセス
3. 共有ボタン → 「ホーム画面に追加」

### PCブラウザ
1. ファイルを開く（file:// or http://）
2. アドレスバー右の「インストール」アイコンクリック
3. デスクトップアプリとして起動可能

⚠ **重要**：file:// プロトコルだとService Workerが動かない（PWA機能制限あり）。  
ローカルHTTPサーバ経由（127.0.0.1:11219等）を推奨。

## 🔑 AI機能の使い方

### Step 1: APIキー取得
1. https://console.anthropic.com にログイン
2. 「API Keys」セクションで新規キー作成
3. `sk-ant-` で始まるキーをコピー

### Step 2: アプリに設定
1. AIタブを開く
2. 「⚙ 設定」をタップ
3. APIキーを貼り付け
4. モデル選択（Haiku 4.5 推奨・月150円）
5. 「設定を保存」

### Step 3: 使う
- **報告ボタン**：記録タブで数字入力後、「★ AIに今日の数字を報告」
- **クイック相談**：AIタブのボタン1タップ
- **自由相談**：AIタブのテキスト欄に質問

## 🔧 トラブルシューティング

### アプリが開かない
→ 4ファイル全部同じフォルダにあるか確認

### Service Worker エラー
→ file:// では動きません、http:// 経由でアクセス

### AI が応答しない
→ APIキー再確認、残高確認（console.anthropic.com）

### データが消えた
→ Chromeデータクリアが原因。JSON エクスポートで復元可能

### グラフが表示されない
→ データが2件以下の時は表示されません

## 🎨 アイコン差し替え

現在の `icon.svg` は暫定版。  
Junさんが新アイコン渡してくれたら差し替え。

2. `manifest.json` の `icons` セクションを更新
3. アプリ再起動
